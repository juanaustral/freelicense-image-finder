# FreeLicense Image Finder

Busca imágenes con licencia **CC0** y **Dominio Público** usando la API de [Openverse](https://api.openverse.org/).

🔗 **Demo:** https://juanaustral.github.io/freelicense-image-finder/

## 🚀 Cómo funciona

1. Escribís un término de búsqueda (en cualquier idioma)
2. Seleccionás filtro: CC0, Public Domain o ambos
3. Resultados en grilla con vista previa, autor, licencia y enlaces

Cada imagen tiene 3 acciones:
- **👁** Ver página de origen
- **🔗** Abrir URL directa de la imagen
- **⬇** Descargar

## 🛠️ Stack

- HTML / CSS / JS vanilla — sin frameworks
- API: [Openverse](https://api.openverse.org/) (motor de búsqueda de imágenes libres)
- i18n: EN / ES
- Hosting: GitHub Pages
